# PorongPorong
마르코브 체인을 활용한 단어 맞추기 및 문장 맞추기 게임
(2019년 겨울 시그)

## 주제
유아용 간단한 게임 제작

## 핵심 기능
> 마르코브 체인을 활용한 문장 제작
> 단어 맞추기
> 문장 맞추기
